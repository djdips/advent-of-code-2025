# Advent of Code 2025

Solutions for Advent of Code 2025 in multiple languages.

## Structure
- `inputs/` → puzzle inputs
- `python/` → Python solutions
- `javascript/` → Node.js solutions
- `typescript/` → TypeScript solutions
- `java/` → Java solutions
- `go/` → Go solutions

Each day follows the pattern:
- `day01.py`, `day01.js`, `day01.ts`, `Day01.java`, `day01.go`
- Input stored in `inputs/day01.txt`
